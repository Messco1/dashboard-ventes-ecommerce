# Dashboard Ventes E-commerce 2024

Analyse des ventes e-commerce avec Python, SQL et Power BI

# Objectif
Analyse des ventes d'une boutique e-commerce fictive 
pour identifier les tendances et KPIs business.

# Outils utilisés
- Python (Pandas) — nettoyage et préparation des données
- SQL (SQLite) — requêtes d'agrégation et analyse
- Power BI — dashboard interactif

# Insights clés
- Electronics représente 43% du CA total (34 982 $)
- North America génère 45% du CA avec le panier moyen 
  le plus élevé (460 $)
- Pic de ventes en janvier, baisse estivale juin-août
- La carte de crédit représente 63% des transactions

# Structure du projet
- `analyse_ventes.ipynb` — notebook Python complet
- `ventes_nettoyees.csv` — dataset nettoyé
- `Dashboard_Ventes_2024.pbix` — fichier Power BI
